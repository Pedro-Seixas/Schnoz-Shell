# Schnoz-Shell
A C/C++ Linux Shell.

Run "make schnoz" to make files.

## Features
- Shell format: user:directory >>> "Prompt here"
- "cd" change current directory
- "&" background process command
- "|" pipe command
- Message of the day displayed when shell is initiated.
